# *Crowdfunding Analytics Project*

## *Overview*
Analyzed crowdfunding campaign data to understand funding patterns, project performance, and key success factors using *Excel, SQL, Tableau, and Power BI*.

## *Objectives*
* Analyze funding trends and campaign performance  
* Identify factors influencing project success  
* Study category-wise and time-based patterns  
* Understand backer engagement  

## *Key KPIs*
* Total Projects & Total Funding  
* Successful vs Failed Projects  
* Category-wise Performance  
* Funding Trends Over Time  
* Top Projects by Backers & Amount Raised  

## *Tools Used*
*Excel | SQL | Tableau | Power BI*

## *Project Files*
📂 Project files & Dashboards: https://drive.google.com/drive/folders/1cFoC3TFQQnbry1vICrSn1G1iYFrnGqM8?usp=sharing

## *Insights*
* Success rates vary across categories  
* Campaign timing impacts funding outcomes  
* Higher backer engagement improves success  

## *Conclusion*
Provides insights into *campaign performance and success drivers*, enabling better crowdfunding strategies.
